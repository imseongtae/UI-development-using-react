## Install react

```bash
$ yarn add react react-dom prop-types
```

## Install babel

```bash
yarn add -D babel-loader @babel/core @babel/preset-env @babel/preset-react
```


## Install webpack

```bash
$ yarn add -D webpack webpack-cli webpack-dev-server clean-webpack-plugin
```
