# Webpack with TypeScript


## Install React

```bash
$ yarn add react react-dom
```

```bash
$ yarn add -D @types/react @types/react-dom
```

## Install TypeScript

```bash
$ yarn add -D typescript
```

```bash
$ npx tsc --init
```

## Install Webpack

```bash
$ yarn add -D  ts-loader webpack webpack-cli webpack-dev-server
```
